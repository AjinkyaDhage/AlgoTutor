Learn A-Star

Go through the following concepts to learn A-Star

1. Algorithm Introduction

A-star Algorithm which is also known as best-first search algorithm is a complete and optimal algorithm, used for solving problems by searching in Artificial Intelligence. To understand the working of A-star algorithm, let us first understand what solving problems by searching is [1].

2. Solving Problems

A problem in AI is formulated using following metrics:

• States: States describe the environment and the agent attributes in which an agent is acting upon to solve the given problem.
• Initial State: This is the state of the environment and an agent in the beginning.
• Goal state: The final state where the agent aims to reach.
• Actions: The actions that an agent can perform to reach to the goal state.
• Path cost: The cost corresponding to each action of an agent.

Goal of the agent is to reach the goal state using allowed actions optimizing the cost function.

3. Strategies

There are following two strategies for solving problems by searching.

1) Uninformed Search Strategies:

These strategies are also called as blind search strategies, as the agent does not have any additional information about the states, other than the problem definition. The agent does not have ability to decide if one non-goal state is more efficient than another. Hence, these strategies keep on generating the successor nodes until they reach goal state. The agent may get stuck in the infinite loop because of this approach. Breadth First Search (BFS), Depth First Search (DFS), Depth Limited Search, Uniform Cost Search, Iterative Deepening Search and Bidirectional Search are the uninformed search strategies.

2) Informed or Heuristic Search Strategies:

In Informed Search strategies, the agent expands the current node based on an Evaluation Function f(n), optimizing the cost of the search. Hence, the node with the lowest cost value is expanded first. These strategies are also called as Heuristic Search Strategies as they use a heuristic function h(n), as a component of function f, which gives the estimated cost of the cheapest path from the current state to the goal state. Greedy Best First Search, Recursive Best First Search (RBFS) and A-star Search are the examples of Informed Search Strategies.

4. Concepts and Example

A-star is a complete and optimal algorithm used for solving problems by searching. It uses function f(n) which is combination of a function g(n), that gives cost to reach the node and a heuristic function h(n), which is the cost to reach the goal state from the node, to evaluate a current node represented as follows.

F(n) = g(n) + h(n)

A-star algorithm performs repeated state checking, assuming that the search space is a graph. This helps to prevent the agent from entering infinite loop. To achieve this a CLOSED LIST (list of nodes that are already expanded) and OPEN LIST (list of nodes that need to be expanded) is implemented and updated during each iteration.

Example

The Heuristic used for the below example and algorithm [2] is Manhattan Distance. Manhattan distance is the direct distance from a node to the goal state. Hence, it is assumed that there are no obstacles (wall) between node and a goal state while calculating Manhattan distance. Hence, for the start position of the brick, the heuristic value is h = 3, as in 3 moves it will reach the goal state if the wall is not present in between. If h represents Manhattan distance, and g represents the distance of node from start or root node, then in each iteration f = g + h is calculated and node with minimum f is considered as the best node.

Algorithm:

 Start.g = 0;
 Start.h = heuristic(Start)
 OPEN	 = [Start];
 CLOSED	 = []
 
 WHILE OPEN is not empty
 N= OPEN.removeLowestF()
 
 IF goal(N)
 RETURN path to N
 
 CLOSED.add(N)
 FOR all children M of N
 not in CLOSED:
 M.parent	= N
 M.g		= N.g+ 1;
 M.h		= heuristic(M)
 OPEN.add(M)
 END FOR

 END WHILE

5. Applications

There are various applications of A-Star Algorithm. As the A-Star Algorithm is capable of finding the cheapest or the shortest path, generally it is used to solve search problems in which an agent has to find the most efficient path to reach the goal state. AlgoTutor uses A-Star Algorithm to solve sliding brick puzzle which is one of its applications. In this puzzle, the goal is to move the master brick to its goal position, but to achieve this, other obstacle bricks are to be moved in such a way that it will be easy for master brick to reach the goal state.

This algorithm can be used as a path finding tools in various games, map applications, and robotics as well. Check the following videos in which the A-Star is used to implement the path finder for a game and for a robotic vehicle.

How to use AlgoTutor Application?

Home Page:

Once you open the application, you will be on the Home page. You can click on GET STARTED to being. You will be navigated to Main Menu page once you click on GET STARTED. To begin, you can also click on the menu icon, which is on the extreme right corner of the application header. Then click on Main Menu to proceed.

Main Menu:

Learn Algorithm, Implement and Execute Algorithm, and Assessment Quiz are the three main features of the application. The application helps to understand A-Star Algorithm. To learn this algorithm in a better way, it is advisable that you begin with Lean section and then proceed to Execute and Implement Section. Finally, take the assessment quiz. Click on Learn A-Star Algorithm to begin the learning process.

Learn A-Star Algorithm:

This section will help you to learn the A-Star algorithm and it assists the users to understand the basics of this algorithm. Go through the given topics sequentially by clicking on topic names, read the concepts and follow the given instructions. Once you understand all the concepts then click on Proceed To Implement and Execute A-Star.

Implement and Execute A-Star:

This section provides the code used for implementing A-Star Algorithm to solve Sliding Brick Puzzle. Click on Implement A-Star to get this code. Read the description of the code to understand the functionality achieved by each code snippet.

This section also provides the steps to solve the Sliding Brick Puzzle that are generated after executing the given code. Click on “Solve Sliding Brick Puzzle” and then click on each puzzle to get the steps to solve it. These steps will be generated by the code provided as explained above.

Once you completely understand the A-Star Algorithm, you can click on Proceed To Assessment Quiz to take the quiz and to confirm the level of your understanding.

Take Assessment Quiz:

Read the quiz instructions and click on Start Quiz, to begin with the quiz. Read each question and select appropriate answer option and then click Next to proceed to the next question. Once you answer all the questions you will receive the score and the result of the assessment quiz. You can click on the View Solution drop-down to get the solution and the description of correct answers.

Additional Navigation Options:

Footer of the application has navigation buttons that will help you to navigate to the required page. You can also use the menu button present on the extreme right corner of the application header to navigate to the required page.
